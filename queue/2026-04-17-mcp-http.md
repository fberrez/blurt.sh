New in Blurt: streamable HTTP transport for MCP.

Your VPS runs the Blurt server AND the MCP endpoint. Your laptop's Claude Code connects to it remotely.

No SSH tunnels, no stdio-over-ssh hacks. Just:

    mcp: { url: https://blurt.your-vps.com/mcp }
